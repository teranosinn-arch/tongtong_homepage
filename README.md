# 🪐 惑星トントン 公式ホームページ 🪐

東京を拠点に活動しているバンド「惑星トントン」の2000年代前半テイストなチープホームページです。

## 🎨 特徴

このホームページは、2000年代前半のインターネット文化を徹底的に再現しています：

### 🌈 ビジュアル要素
- **派手なカラースキーム** - 蛍光色と原色を大量に使用
- **グラデーションアニメーション** - 常に色が変わるヘッダーとフッター
- **キラキラエフェクト** - 背景全体にスパークル効果
- **テーブルレイアウト** - CSS Grid/Flexboxで再現した懐かしのテーブル設計
- **テキストシャドウ** - 多重の色付きシャドウで立体感を表現

### 🎯 インタラクティブ要素
- **マウスクリックエフェクト** - クリック時にキラキラが飛び散る
- **ホバーアニメーション** - ボタンなどが拡大・回転する
- **コナミコード** - ↑↑↓↓←→←→BA で特別なイースター卵が発動
- **マウストレイル** - マウスカーソルの軌跡にパーティクル
- **スムーススクロール** - ナビゲーション連動のスクロール

### 🎭 レトロ要素
- **Comic Sansフォント** - 2000年代の典型的なフォント
- **UNDER CONSTRUCTION バナー** - 建設中のアニメーション表示
- **ドットカウンター** - 訪問者カウンター（Web Counter）
- **Flash的なアニメーション** - CSSで実現した懐かしのアニメーション
- **GIF風の回転** - HTML5での再現

## 🎮 イースター卵

1. **クリックエフェクト** - ページをクリックするとキラキラが飛び散る
2. **コナミコード** - ↑↑↓↓←→←→BA キーを順番に押すと特別なメッセージが表示される
3. **マウストレイル** - マウスカーソルを動かすと虹色のトレイルが表示される
4. **コンソール** - ブラウザの開発者ツール（F12）を開くと特別なメッセージが表示される

## 🔧 カスタマイズ

### 色を変更したい場合
`style.css` の以下の部分を編集してください：

```css
background: 
    repeating-linear-gradient(
        45deg,
        #FF00FF,  /* ここの色を変更 */
        #FF00FF 10px,
        #00FFFF,  /* ここの色��変更 */
        ...
    )
```

### フォントを変更したい場合
`style.css` の `font-family` を変更するか、Google Fonts から新しいフォントを追加してください。

### アニメーション速度を調整したい場合
`@keyframes` のセクションで `animation: XXs` の値を変更してください。
